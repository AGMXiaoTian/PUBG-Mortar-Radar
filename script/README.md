# PUBG 迫击炮测距脚本 (Python 后端)

基于屏幕颜色识别的 PUBG 外部迫击炮测距脚本。识别地图上的玩家标点，结合游戏迫击炮物理弹道公式，自动计算并在屏幕上绘制迫击炮开火密位。

## 核心功能

- **多目标同屏追踪**：同时追踪最多 4 个队伍标点（黄/红/蓝/绿）
- **有效射程过滤**：自动隐藏距离小于 100 米或大于 700 米的无效数据
- **大地图/小地图双模式**：支持小地图后台静默测距 + 大地图 (F9) 一键宏缩放测距
- **可视化调试面板**：PyQt6 GUI 配置工具，支持自定义分辨率、色盲模式、扫描频率 (FPS)
- **攻城模式 (F8)**：中心垂直标尺，显示各刻度对应的开火密位

## 运行环境

1. **显示模式**：游戏必须设置为 **无边框窗口模式 (Borderless Windowed)**
2. **运行**：管理员模式运行 `radar_scanner.py`

## 安装

```bash
pip install -r requirements.txt
```

## 配置

1. 复制 `config.example.json` 为 `config.json`
2. 运行 `python config_tool.py`（tkinter 版）或 `python ui_config_tool.py`（PyQt6 版）进行屏幕区域校准
3. 使用截图工具精确测量小地图区域坐标和玩家白色箭头中心点坐标

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| **F8** | 开启/关闭中心垂直标尺 |
| **F9** | 大地图宏指令测距 |
| **N** | 切换小地图/小小图识别模式 |
| **Shift+N** | 仅切换游戏地图，不切换脚本模式 |

## 常见问题

**Q: 按了 F8 有标尺但没有数字？**
- 标点距离小于 100 米或大于 700 米被自动过滤
- 色盲模式与配置不一致
- 游戏 UI 比例特殊，需重新校准截图区域

**Q: 呼出标尺时游戏卡顿？**
- 确认游戏处于无边框模式
- 尝试右键游戏程序 → 属性 → 兼容性 → 勾选"禁用全屏优化"

**Q: 程序闪退？**
- 检查同目录下是否存在 `config.json`。先运行配置工具保存生成该文件
